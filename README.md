                          Project Overview

Healthy Grocery List Planner is a web application that helps users create and manage grocery lists for healthy eating.  
Users can add products with quantities, group them by categories, mark items as purchased, and manage their lists securely after authentication.  
The project is built using ASP.NET Core MVC and follows clean architecture principles.


            
     
                         Project Features 

1. User registration and login
2. Create grocery lists
3. Add / edit / delete products
4. Group products by category
5. Mark items as purchased
6. View grocery list
7. Access grocery data via API

### 1. Login / Register Screen
This screen allows users to log in or create a new account.
**Screen elements:**

- Password input field

### 2. Grocery List Screen
This screen displays the user's grocery list with products grouped by category.
**Screen elements:**
- Grocery list title  
- Products grouped by category  
- Checkbox to mark products as purchased  
- Add Product button  
- Logout button  

### 3. Add Product Screen
This screen allows users to add a new product to the grocery list.
**Screen elements:**
- Product name input field  
- Quantity input field  
- Category input field  
- Save Product button  
- Cancel button  

                                Application Workflow
                                
1. User opens the application  
2. User logs in or registers  
3. User is redirected to the Grocery List screen  
4. User adds products to the list  
5. User marks products as purchased  
6. User edits or deletes products if needed  
7. User logs out
